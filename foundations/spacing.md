# Spacing Foundation

Spacing should create calm structure and scannability.

## Principle

Use generous spacing for executive readability. Avoid cramped layouts and dense visual systems unless the content is intentionally data-heavy.

## Scale

| Token | Use |
|---|---|
| `--space-xs` | Tight inline spacing |
| `--space-sm` | Small component spacing |
| `--space-md` | Standard component spacing |
| `--space-lg` | Card and module padding |
| `--space-xl` | Section spacing |
| `--space-2xl` | Large section breaks |
| `--space-3xl` | Hero and page-level rhythm |
