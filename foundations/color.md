# Color Foundation

The starter palette is built around a serious executive system: dark navy, structured slate, restrained amber, and a secondary teal.

## Token roles

| Token | Role |
|---|---|
| `--color-brand-navy` | Primary background and executive-brand anchor |
| `--color-brand-slate` | Elevated surface and card background |
| `--color-brand-amber` | Accent, highlight, and callout color |
| `--color-brand-teal` | Secondary accent for systems, status, or workflow moments |
| `--color-text-primary` | Main text on dark backgrounds |
| `--color-text-muted` | Secondary and explanatory text |
| `--color-border-default` | Structural borders and dividers |

## Usage principle

Use color to clarify hierarchy, not decorate the interface.
