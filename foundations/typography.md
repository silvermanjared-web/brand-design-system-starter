# Typography Foundation

The default type system uses Inter or a system sans-serif fallback.

## Voice in interface

The typography should feel:

- clear
- senior
- modern
- calm
- readable
- not ornamental

## Scale

| Token | Use |
|---|---|
| `--font-size-sm` | Labels, metadata, helper text |
| `--font-size-base` | Body text |
| `--font-size-lg` | Lead text |
| `--font-size-xl` | Section headings |
| `--font-size-2xl` | Page headings |
| `--font-size-3xl` | Hero headlines |
