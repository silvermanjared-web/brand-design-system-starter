# Security Policy

## Supported scope

This is a public design-system starter. It should not contain private client files, proprietary brand assets, credentials, paid font files, or non-public source code.

## Please report

Please open an issue if you find:

- exposed credentials or tokens
- private client or company data
- paid font files or proprietary binary assets
- confidential design files
- unsafe instructions that encourage committing private material
- examples that appear to include real customer, campaign, or account data

## Public-safe usage

Use placeholder assets, public examples, and synthetic content in this repo.

Do not commit:

- private Figma files
- paid font binaries
- client logos without rights
- private screenshots
- customer data
- platform exports
- API keys
- OAuth tokens
- production secrets

## Intended use

This repo is intended to demonstrate design-system structure, token thinking, component documentation, and AI handoff patterns. It is not a place to store confidential brand or client materials.
